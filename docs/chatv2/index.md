## User Chat

### Core Features
- Send and receive messages
- Chat listing with:
  - Last message
  - Last message status (single tick, double tick, blue tick)
  - Time
- Chat history with search
- Message reply
- Send files
- Mute / unmute chat
- Block / unblock user

### Real-time Features
- Typing indicator
- Message status updates (sent, delivered, seen)
- Online / offline status (green dot)
- Real-time unread count
- Multi-device support (same user on multiple devices)

### Offline Handling
- Undelivered messages auto-delivered when user comes online
- Last 3 unread message notifications sent on reconnect
